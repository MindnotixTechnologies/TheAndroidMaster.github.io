A material design Spotify client for Android.

Status is a material design status bar replacement for android that can function properly without needing root permission.
It was created for the main purpose of reducing the amount of space taken up by status icons on my phone, which I am unable to gain root access on.

Design

  • All icons take up the smallest amount of space possible while still providing a clear message.
  • The status bar background will fade between color changes.
  • When fullscreen mode is toggled, the status bar animates off the screen.

Features

  • Change the color of the status bar in specific apps
  • Changes the icon color based on the color of the status bar
  • Choose whether to display the AM/PM indicator in the clock
  • An option to show the battery percentage alongside the icon

Limitations

  • Unfortunately, there is no way to make this status bar fully transparent, since it is only an overlay - the system status bar will be visible underneath it
  • It is difficult to anticipate all movements of the system status bar, for example the app currently cannot detect a swipe down from the top of the screen in fullscreen mode. There are also bound to be some device-specific gestures that cannot be accounted for *cough cough Samsung
